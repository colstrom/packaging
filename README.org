* command/get-pkgsrc

  Configures a system to use pkgsrc.

** Usage

   #+BEGIN_SRC shell
     git clone https://github.com/colstrom/packaging
     cd packaging
     command/get-pkgsrc
   #+END_SRC

   Optionally, set =MIRROR= to your package host. It defaults to my testing
   repository at =https://unpkg.me=.
